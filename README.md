# 🐍 Snake: Modern Edition

Современная реализация классической игры "Змейка" на Python с использованием библиотеки **Arcade (3.x)**.

Проект отличается стильным минималистичным дизайном, плавной анимацией, интерактивным меню и гибкой настройкой сложности.

## ✨ Особенности

- 🎨 **Современная графика**: Тёмная тема, неоновые акценты, градиентное тело змейки и эффекты свечения.
- 🎬 **Плавная анимация**: Змейка скользит между клетками (интерполяция движения), кнопки имеют эффект увеличения при наведении, заголовок пульсирует.
- 🎮 **Геймплей**:
  - Три уровня сложности (Easy, Medium, Hard).
  - Динамическое увеличение скорости игры по мере роста счёта.
  - Сохранение лучшего результата (High Score) между сессиями.
- 🖥️ **Удобный интерфейс**: Стартовое меню, экран Game Over, подсказки управления.
- ⚡ **Производительность**: Оптимизированный рендеринг текста и объектов для Arcade 3.3.3+.

## 📦 Установка и запуск

### Требования
- Python 3.8+
- Библиотека `arcade` (версия 3.3.3 или новее)

### Инструкция

1. **Создайте папку проекта:**
   ```bash
   mkdir snake_game
   cd snake_game
   ```

2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv venv
   ```

3. **Активируйте окружение:**
   - Windows (PowerShell): `.\venv\Scripts\Activate.ps1`
   - Linux/macOS: `source venv/bin/activate`

4. **Установите зависимости:**
   ```bash
   pip install arcade
   ```

5. **Запустите игру:**
   ```bash
   python main.py
   ```

## 🕹️ Управление

| Клавиша | Действие |
| :--- | :--- |
| **W / ↑** | Вверх |
| **A / ←** | Влево |
| **S / ↓** | Вниз |
| **D / →** | Вправо |
| **ESC** | Вернуться в меню |
| **ENTER** | Начать игру (в меню) |
| **Мышь** | Навигация в меню и выбор сложности |

## 📂 Структура проекта

Проект имеет модульную архитектуру для легкого расширения:

```
snake_game/
├── main.py          # Точка входа (запуск окна)
├── config.py        # Глобальные настройки (цвета, размеры, шрифты)
├── game.py          # Главный игровой цикл и логика состояний
├── snake.py         # Логика и отрисовка змейки
├── food.py          # Логика и анимация еды
├── ui.py            # Интерфейс (меню, кнопки, HUD)
├── README.md        # Документация
└── high_score.json  # Автоматически создается при игре (рекорд)
```

## 🛠️ Технологии

- **Python 3**: Язык программирования.
- **Arcade 3.3.3**: Библиотека для 2D-игр. Используется современное API (`draw_lbwh_rectangle_filled`, `arcade.Text` для производительности).
- **JSON**: Для сохранения локального рекорда.
