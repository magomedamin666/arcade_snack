
# 🐍 Snake: Modern Edition

Современная реализация классической игры "Змейка" на Python с использованием библиотеки **Arcade (3.x)**.

Проект отличается стильным минималистичным дизайном, плавной анимацией, интерактивным меню и гибкой настройкой сложности.

---

## ✨ Особенности

* 🎨 **Современная графика**: Тёмная тема, неоновые акценты, градиентное тело змейки и эффекты свечения
* 🎬 **Плавная анимация**: Интерполяция движения, hover-эффекты кнопок, пульсация заголовка
* 🎮 **Геймплей**:

  * Три уровня сложности (Easy, Medium, Hard)
  * Увеличение скорости со временем
  * Сохранение лучшего результата (High Score)
* 🖥️ **Интерфейс**: Стартовое меню, экран Game Over, удобная навигация
* ⚡ **Производительность**: Оптимизированный рендеринг под Arcade 3.x

---

## 📦 Установка и запуск

### Требования

* Python 3.8+
* Библиотека `arcade` (3.3.3+)

### Установка

```bash
mkdir snake_game
cd snake_game
python -m venv venv
```

### Активация окружения

**Windows (PowerShell):**

```bash
.\venv\Scripts\Activate.ps1
```

**Linux/macOS:**

```bash
source venv/bin/activate
```

### Установка зависимостей

```bash
pip install arcade
```

### Запуск

```bash
python main.py
```

## 🕹️ Управление

| Клавиша | Действие        |
| ------- | --------------- |
| W / ↑   | Вверх           |
| A / ←   | Влево           |
| S / ↓   | Вниз            |
| D / →   | Вправо          |
| ESC     | В меню          |
| ENTER   | Начать игру     |
